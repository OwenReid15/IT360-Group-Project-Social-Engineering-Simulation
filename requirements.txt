Requirements

Operating System:
Kali Linux (recommended) or any Debian-based Linux distribution.

Installed Tools:
Hashdeep
Bulk Extractor
ExifTool

Install all required tools using:
sudo apt install bulk-extractor hashdeep exiftool

Bash:
Bash shell must be installed (included by default on Kali).

Permissions:
User must have permission to read the evidence file and write to the output directory.
